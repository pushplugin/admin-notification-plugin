The push notification plugin for wordpress admins.

== Description ==

This plugin helps to send push notification to wordpress admin whenever any of the user perfoms an action on post/page or comments.
Admin can subscribe to the events and can receive push notification whenever that event occurs.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `pushplugin-admin-notification.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can we collect subscribers so that we can push notification to them? =

No, this plugin is only for sending push notification to admin when ever an event occurs and admin has subscribed to that event.

= How can we collect subscribers then to send them push notifications? =

You can use our other plugin [PushPlugin](https://pushplugin.com/) to collect subscribers and send push notifications to them.
